Pixelating Process

I used audacity to glitch the pixelated images. From there, I compiled the different versions of each image in its own respective gif. Flowing from the biggest pixel size of 60, then its glitched version, all the way to the original image.


Glitching with Fade out Filter on Audacity with pixel size 60 images

Represents me "fading out" during working. Since these images had the smallest pixel size that I used, I wanted just little artifacts in the glitches so that it transitions into the original image in the gif.

Glitching with Delay Filter on Audacity with pixel size 40 images

Represents the delayed reactions and realizations after long periods of working. Not quite the full image, but closer with the colors and smaller pixels.

Glitching with Phaser Filter on Audacity with pixel size 20 images

Represents the millions of things going on during finals season to make me overwhelmed/overstimulated. Using "phaser" describes the speed at which my brain and thoughts are moving. It is difficult to see what the picture looks like at this point.
