DIY Photoshop

For this project, I creating drawing tools that represent a Tsuro game board. Using the #1 key will allow you to place the background that will help you to center all of the pieces. #2-8 are the player keys. #9 clears the canvas if you need to start over. #0 does a screenshot of the current state of the canvas. The letter keys are all configured to be the background pieces that connect to other pieces.
